## 📦 运行

1. 修改环境变量 VITE_authBaseUrl，VITE_dataBaseUrl (.env)为 startingmultitenant 项目的ip和端口

2. 安装依赖

   项目根目录下执行
   ```
   yarn install
   ```

3. 测试运行

    项目根目录下执行后，访问localhost:5173，登录管理员账号
    ```
    yarn dev
    ```

4. build

   ```
   yarn build
   ```
